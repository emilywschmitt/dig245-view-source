How to Use Dev Tools

If you've ever used the internet (I know you have because you're reading this right now),
you've probably come across some cool websites or pages and wondered "how did they make that???"
Well, with Dev Tools, you can find out. Diving into the source code behind your favorite websites can
teach you a ton about coding and design and it's easy to do! Just follow these simple steps.

  1. This can be done on any website, but you must use Chrome as your browser.
  2. Navigate to the webpage and right-click anywhere to open up your menu.
    Be sure to click somewhere that won't take you anywhere else on the page, like a button or shopping cart.
  3. Once you're in the menu, click on 'inspect.'
  4. A window will pop up on the right-hand side of your screen. You are now inside that website's source code!
    If you want to learn about some more thing syou can dow with Dev tools, check out our list below!

Other Cool Things to Do With Dev Tools

  • CSS Selectors. Get the CSS selector from an element if you're having trouble specifying in CSS.
  • Edit as HTML. This is a good way to test changes in code by seeing what visually changes when you edit the HTML.
  • Scroll into view. This is an easy way to naviagte the website within Dev Tools by bringdifferent elemetns into the viewport.

A New Metaphor

There's nothing quite like a great metaphor...

Using Dev Tools is turning over a picture to find the name, location, and date handwritten on the back of the photo.
  • Say your flipping thorugh an old fmaily scrapbook. The pictures bring back good memories and the group pictures remind you of your large

  • You slide the photo out of its sleeve and find, written on the back in your grandmother's lovely cursive writing, the date, location, and
    name of every person in the photo.
  • While revelaing the source code behind a website may not bring back such nostaligic memories, what it does do is answer questions
    just as the writing on the back of a photo does: when it was made, who made it, who helped make it, and how it came to be.
